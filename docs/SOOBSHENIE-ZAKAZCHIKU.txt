Домен зарегистрировали — отлично. Дальше нужно привязать apexgard.ru к готовому сайту. Это делается через DNS в Selectel, сервер покупать не нужно.

Что нужно сделать вам (5–10 минут):

1. Зайти в my.selectel.ru → Домены → apexgard.ru → DNS-записи

2. Добавить 4 A-записи для @ (корень домена):
   185.199.108.153
   185.199.109.153
   185.199.110.153
   185.199.111.153

3. Добавить CNAME для www → likeavenus.github.io

4. Удалить старые A/CNAME для @ и www, если они ведут на заглушку Selectel

5. Написать мне: «DNS добавил»

Полный доступ к Selectel не нужен — достаточно этих записей.

Подробная инструкция с таблицами: файл INSTRUKCIYA-ZAKAZCHIKU.md (можем скинуть PDF/скрин).

После DNS я с своей стороны:
— подключу домен к сайту и HTTPS
— сделаю QR-код на apexgard.ru
— подключу Яндекс Метрику, когда пришлёте код счётчика (для статистики переходов с QR)

Обычно сайт начинает открываться за 1–2 часа после DNS.
