Привет!

Чтобы сайт открывался на apexgard.ru и полностью был на вашей стороне (не на GitHub разработчика), нужен свой небольшой сервер в Selectel.

Что сделать вам:

1. Selectel → Облачная платформа → Создать сервер
   • Ubuntu 24.04
   • 1 vCPU, 1 GB RAM, 10 GB — этого хватит
   • Записать IP сервера

2. Домены → apexgard.ru → DNS:
   • A запись @ → IP сервера
   • A запись www → IP сервера

3. Дать разработчику SSH-доступ на этот сервер:
   • IP, логин (root или deploy)
   • Добавить SSH-ключ разработчика (пришлёт) ИЛИ временный пароль

Полный доступ к аккаунту Selectel не нужен — только один сервер.

После этого разработчик:
— поставит сайт и HTTPS
— проверит apexgard.ru
— отдаст QR и подключит Метрику (когда пришлёте код)

Подробная инструкция с таблицами — файл INSTRUKCIYA-SERVER-ZAKAZCHIKU.md

Примерная цена сервера: от ~300–600 ₽/мес. Домен у вас уже есть.

Напишите, когда сервер создан и DNS прописан — «готово, IP такой-то».
