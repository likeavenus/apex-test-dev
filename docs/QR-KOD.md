# QR-код для APEXGARD

## Ссылка для QR (со статистикой в Метрике)

```
https://apexgard.ru/?utm_source=qr&utm_medium=offline&utm_campaign=apexgard
```

После подключения Яндекс Метрики переходы с QR будут видны в отчётах:
**Отчёты → Источники → UTM → utm_source = qr**

## Быстро сгенерировать QR

1. Откройте https://www.qr-code-generator.com/ или https://metrika.yandex.ru (раздел QR, если доступен)
2. Вставьте ссылку выше
3. Скачайте PNG/SVG для печати

## После деплоя на домен

Разработчик добавит файл `assets/qr-apexgard.png` в репозиторий.
